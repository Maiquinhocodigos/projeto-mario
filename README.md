# projeto-mario
Projeto da criação de um 'site' simples.

O site foi feito para colocar na pratica as linguagens e ferramentas, e também melhorar técnicas em 'front-end'.

tentei explorar ao máximo as principais tags do HTML, assim como deixar o acesso em todas as plataformas através da responsividade do site
algumas tags foram customizadas no site, porém sem uma função especifica como o "HOME" já que o site só tem 1 tela.

Esse foi meu primeiro contato com HTML, CSS e JavaScript. Também tive meu primeiro contato com responsividade através desse projeto
